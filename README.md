# Savings Tracker

A modern savings tracker built using HTML, CSS, and JavaScript that helps users manage savings goals and track progress efficiently. The application provides a simple interface for adding savings, updating balances, viewing progress, and maintaining a history of savings activity.

## Features

- Add and track savings amounts
- Edit current savings balance
- Set and update savings goals
- Dynamic progress bar with live updates
- Remaining amount calculation
- Savings history with timestamps
- Total activity/day tracking
- Data persistence using Local Storage
- Responsive and clean user interface

## Tech Stack

- HTML
- CSS
- JavaScript
- Local Storage API

## How It Works

1. Users enter a savings amount and add it to the tracker.
2. Savings progress updates automatically.
3. The app calculates the remaining amount needed to reach the goal.
4. History entries are stored with date and time information.
5. All data is saved using Local Storage so progress remains after refreshing the page.
